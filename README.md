## Deployment

### Vercel (Khuyến nghị)

1. Tải file `thung-carton-production.tar.gz`
2. Giải nén và upload lên Vercel
3. Cấu hình domain: `lading.thungcartonbienhoa.vn`
4. Thêm environment variable: `VITE_GA_MEASUREMENT_ID`

### Cấu trúc thư mục

```
├── public/
│   ├── hero-video-optimized.mp4 (23.5MB)
│   ├── images/ (86 sản phẩm)
│   └── favicon.ico
├── src/

```
